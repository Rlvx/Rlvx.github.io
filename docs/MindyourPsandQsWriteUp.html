Explication

Le fichier txt qui nous ai donné nous donne 3 valueur (c,e,n). On a le mot RSA qui nous fait comprendre que ce sont des paramètres RSA, le but est de décrypter le ciphertext c. Pour décrypter c il faut donc d’abord comprendre comment fonctionne RSA et à quoi correspond ces valeurs c,e et n.

Chiffrement RSA

  1. prenons deux nombres premiers ppp et qqq
  2. calculons n=p.qn = p.qn=p.q.
  3. calculons ϕ(n)=(p−1)(q−1)\phi(n) = (p-1)(q-1)ϕ(n)=(p−1)(q−1)
  4. choisissons un eee (exposant publique) premier avec ϕ(n)\phi(n)ϕ(n) (souvent e=65537e = 65537e=65537)
  5. calculons d=inversemod(e,ϕ(n))d = inverse_{mod}(e,\phi(n))d=inversemod​(e,ϕ(n)) (exposant privé)

Une fois toutes ces étapes faites, on peut chiffrer un message mmm en ccc en faisant c=me[n]c=m^{e}[n]c=me[n]

pour récuperer le message mmm il faut être détenteur de la clé privé ddd afin de pouvoir calculer m=cd[n]m=c^{d}[n]m=cd[n]

But du challenge

Dans ce challenge nous somme dans la postion d’une personne ayant intercépté un message c accompagné de e et de n. Nous n’avons pas d pour déchiffrer et nous ne connaissons pas p et q. Cependant une information importante : n est petit. En effet la sécurité de RSA repose sur la difficulté à factorisé n en deux nombre premiers. mais cette difficulté ce retrouve pour des clés très grande (1024 bits). Ici, il est possible de retrouver p et q en utilsant le site factordb

from factordb.factordb import FactorDB
from Crypto.Util.number import long_to_bytes

def egcd(a, b):
    if a == 0:
        return (b, 0, 1)
    else:
        g, y, x = egcd(b % a, a)
        return (g, x - (b // a) * y, y)

def modinv(a, m):
    g, x, y = egcd(a, m)
    if g != 1:
        raise Exception('modular inverse does not exist')
    else:
        return x % m

def RSA_factorize_decrypt(c,n,e)
	f = FactorDB(n)
	f.connect()
	factors = f.get_factor_list()
	p = factors[0]
	q = factors[1]
	phi = (p-int(1))*(q-int(1))

	d=modinv(e,phi)

	m = pow(c,d,n)
	print(long_to_bytes(m))

le programme n’est pas très optimisé car il n’englobe pas certaines particularités, notamment si n est combinaison de plusieurs facteurs ou si n est combinaison du même nombre premier. Néanmoins il marche dans la cadre du challenge
