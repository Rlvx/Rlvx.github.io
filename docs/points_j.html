points_j

Enoncé
Vous assistez à un entretien pour un poste en Forensic et une personne vous propose ce challenge à résoudre le plus vite possible (en ayant Internet).
Il vous demande de trouver la date d’impression ainsi que le numéro de série de l’imprimante cachés dans ce document.
Vous restez dubitatif et acceptez le challenge.

La réponse est sous la forme :
hh:mm dd/MM/yyyy SSSSSSSS

avec SSSSSSSS : le numéro de série

Explication
Voici un site sympathique pour analyser des images (autre qu’Aperisolve) :
https://29a.ch/photo-forensics/#forensic-magnifier

en faisant charger l’image, avec l’option
enhancement - Auto contrast by channel
on peut passer sur le document et apercevoir les points jaunes

Solution.
On passe le zoom dessus en calculant les points grace a une documentation dans

https://www.eff.org/files/filenode/printers/ccc.pdf
