Description initiale :

The numbers… what do they mean?

Explications

chaque nombre correspond au numero d’une lettre dans l’alphabet

1 = A
2 = B
…

pour déchiffrer ce message rien de très compliqué. Il est possible de résoudre ce challenge à la main mais je préfère coder un code python qui déchiffre ça automatiquement.

Code python

def thenumbers_decrypt(text):  
  alphabet = list("abcdefghijklmnopqrstuvwxyz")  
  buffer=""  
  for values in text.split():  
	  if str(values).isnumeric():  
	  buffer+=alphabet[int(values)-1]  
  else:  
	  buffer+=values  
  return buffer

fonction très simple qui va split le texte à chaque espace pour récuperer chaque packets de charactères (ici les nombres), si c’est un nombre j’affiche sa lettre associé, (le -1 dans alphabet[int(values)-1] est du au fait que la lettre A est à l’index 0 de la liste, donc tout est décalé de -1 (A=0,B=1…)

Exectution du programme
