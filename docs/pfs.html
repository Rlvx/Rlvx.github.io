Explication
On le verra assez vite (et merci au créateur du challenge pour l’indice dans le titre, qui a été pour moi immédiatement parlant), les espaces et tabulations du texte sont importants (et les sauts de ligne, histoire de bien faire un langage ésotérique ternaire).

Les espaces entre les caractères sont eux à négliger (j’ai mis un peu de temps à le comprendre)

Solution
1- One-liner
On supprime, dans l’ordre : la ponctuation / les espaces entre chaque caractères (il faut faire cette étape par deux fois via le sed pour bien tout nettoyer) / les caractères alphanumériques.

$ cat ch19.txt | tr -d [:punct:] |sed "s/[[:alnum:]] [[:alnum:]]//g" |sed "s/[[:alnum:]] [[:alnum:]]//g"| tr -d [:alnum:] > ch19.ws

2- Interpréteur
Une fois ceci fait, on installe un petit module python qui va bien :

$ pip3 install whitespace

3- Résolution

$ whitespace ch19.ws
