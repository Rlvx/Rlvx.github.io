Explications

Challenge typique en cryptographie, le ROT ou Cesar. Ici il s’agit d’un ROT13 donc toutes les lettres du flag sont décalé de 13. autrement dis, un A devient un N un B un O etc…

des sites propose de faire des décrytage de Cesar mais je vais plutôt faire mon propre code python.

Code python

def cesar(text, rot):
    result = ""
    for char in text:
        if char.isupper():
            result += chr((ord(char) + rot - 65) % 26 + 65)
        elif char.islower():
            result += chr((ord(char) + rot - 97) % 26 + 97)
        else:
            result += char
    return result

la fonction cesar demande 2 arguments :

  • le texte à chiffrer/déchiffrer
  • la clé de décallage

Résolution

ici la clé est donné dans l’énoncé (ROT13), je lance mon programme avec comme paramètre cvpbPGS{arkg_gvzr_V'yy_gel_2_ebhaqf_bs_ebg13_hyLicInt} et 13 :
